== 功能 ==
在编辑器上添加一个按钮，生成选中文字到萌娘百科的链接。

目测是由互动的百科插件改过来的。

=== 使用方法 ===
1鼠标拖动选中一个需要解释的词（比如萝莉）
2点编辑器上绿色的【萌】字
3指向萌娘百科的链接自动生成了~

=== 作用 ===
？对于一些需要长篇废话解释，或者浏览者大多不熟悉的内容可以直接生成链接，无需每次都解释一遍？
